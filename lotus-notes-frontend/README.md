# Lotus Notes Frontend

Interfaz gráfica para el sistema de colaboración Lotus Notes.

## Instalación

```bash
cd lotus-notes-frontend
npm install
```

## Uso

```bash
npm run dev
```

La aplicación se abrirá en `http://localhost:3000`

## Características

- 🔐 Login y Registro
- 📊 Dashboard con estadísticas
- 📝 Gestión de Notas
- ✅ Gestión de Tareas
- 📅 Calendario de Eventos
- 💬 Sistema de Mensajería

## Requisitos

- El backend debe estar corriendo en `http://localhost:4000`
- Node.js 16+

## Tecnologías

- React 18
- Vite
- React Router
- Axios
